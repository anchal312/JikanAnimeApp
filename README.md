# Jikan Anime Viewer (Android) - Complete Project

This project includes:
- Retrofit for Jikan API calls
- Room for local caching
- Jetpack Compose UI (list + detail)
- WorkManager for periodic sync
- MVVM with ViewModel and StateFlow

How to use:
1. Unzip and open the folder in Android Studio.
2. Gradle should sync (uses Gradle 8.4 and AGP 8.3.2). If not, update your Android Studio to a recent version.
3. Run the app on an emulator or device. Internet permission is included.
